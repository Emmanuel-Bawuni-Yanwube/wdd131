Social Media Hub - Site plan information

The new web pages are intended to use proper techniques learned in class to simplify and improve a site that provides resources and strategies for managing social media handles effectively.

Step 1 - Setup

This file: siteplan.html

Step 2 - Site Content

  1. Site Name: The Social Media Hub
  2. Site Purpose: Describe strategies, tools, and training for growing and managing social media handles. Provide free tips and resources that help users improve their online presence and engagement.
  3. Scenarios:
    • How can I grow my Instagram followers effectively?
    • What tools can help me manage multiple social media platforms?
    • How do I write the perfect bio for my social media profiles?
  4. Color Schema: The colors are defined in the :root element at the top of styles/sunburst.css file. Currently using a vibrant blue for headings and pink accents for call-to-actions. More colors will be added as the project progresses.
  5. Typography:
    Explicitly use Roboto family in both the body and h1-header.

    Preliminary Main and Mobile wireframe images are shown below.
